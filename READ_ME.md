# SCT_WD_2 — Stopwatch Web App

An interactive stopwatch web application built as part of the SkillCraft Technology Web Development Internship.

## Task Description
Create an interactive stopwatch with start, pause, reset, and lap tracking functionality.

## Features
- Start / Pause / Resume / Reset
- Millisecond precision (MM:SS.mmm)
- Animated circular progress ring
- Lap recording with fastest/slowest highlighting
- Clear all laps

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript

## How to Run
Open `index.html` in any browser. No setup needed.
